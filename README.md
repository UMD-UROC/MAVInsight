# MAVInsight

MAVInsight is a tool for live MAVLINK data visualization using MAVROS, PlotJuggler, and QGroundControl. This repository provides a step-by-step guide to set up the required environment for seamless integration and testing.

## Features

- **Live MAVLINK Data Visualization**: Stream and analyze MAVLINK data in real-time.
- **Integration with PX4 and ROS2**: Leverage PX4 and ROS2 for advanced robotics and simulation.
- **Support for PlotJuggler**: Visualize data streams with PlotJuggler for better insights.

## Installation and Setup

Follow the [Installation and Setup Guide](docs/README.md) for detailed instructions on setting up the environment, including:

1. Installing PX4, ROS2, and QGroundControl.
2. Setting up PlotJuggler for live data visualization.
3. Configuring MAVROS for seamless communication.

## Prerequisites

- **Operating System**: Ubuntu 22.04 LTS.
- **Required Tools**: `git`, `wget`, `curl`, `make`, `python3`, and `pip`.
